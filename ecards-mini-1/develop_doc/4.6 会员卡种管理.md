# 会员管理系统 API 接口文档

## 统一返回格式

所有接口均使用 "Result" 包装器返回数据，结构如下：

| 字段名  | 类型   | 说明                                          |
| ------- | ------ | --------------------------------------------- |
| code    | Int    | 状态码，200 表示成功；非 200 为错误码         |
| data    | Object | 响应数据对象，无数据时返回空对象              |
| message | String | 响应消息，成功时返回操作结果描述，失败时返回具体错误信息 |

## 4.6 会员卡种管理模块

### 4.6.0 模块接口列表

| 序号 | 接口名         | 接口 URL                               | HTTP 方法 | 功能描述                                                                 |
| ---- | -------------- | -------------------------------------- | --------- | ------------------------------------------------------------------------ |
| 1    | 会员卡种创建   | /api/v1/member-card-types/create       | POST      | 商家创建会员卡种                                                         |
| 2    | 会员卡种列表查询 | /api/v1/member-card-types/list-query | GET       | 查询指定店铺下所有可用的会员卡种                                         |
| 3    | 会员卡种详情查询 | /api/v1/member-card-types/detail-query | GET     | 查询指定卡种的完整配置信息                                               |
| 4    | 会员卡种修改设置 | /api/v1/member-card-types/set        | POST      | 修改基础信息（卡种名称、描述、通知类型、预设项目、跨店通用），卡种类型不可修改 |

### 4.6.1 接口 1：会员卡种创建

* **接口名**：会员卡种创建（店铺创建专属会员卡种）

* **业务背景**：商家进入店铺后可创建卡种，需设置卡种名称、类型、预设充值/消费项目、通知类型、跨店通用等信息。

* **认证**：仅商家可操作，需携带商家普通令牌（role=MERCHANT，tokenType=1）

* **URL**：/api/v1/member-card-types/create

* **HTTP Method**：POST

* **请求头**：

| 请求头字段     | 是否必填 | 示例值                | 描述           |
| -------------- | -------- | --------------------- | -------------- |
| Content-Type   | 是       | application/json      | 请求体数据格式 |
| Authorization  | 是       | Bearer {普通令牌}     | 商家普通令牌   |

* **请求参数**（请求体）：

| 参数名         | 类型    | 是否必填 | 示例值                                                       | 描述                                           |
| -------------- | ------- | -------- | ------------------------------------------------------------ | ---------------------------------------------- |
| storeId        | String  | 是       | uuid-90ab-cdef-1234-5678                                     | 所属店铺 ID                                    |
| cardTypeName   | String  | 是       | 黄金余额卡                                                   | 卡种名称（长度 2-50 位）                       |
| description    | String  | 否       | 消费满100减10，享专属折扣                                    | 卡种描述（长度 0-500 位）                      |
| cardMask       | String  | 否       | 1 或 /oss/card/mask1.png                                     | 卡面蒙版图片（数字或OSS路径，最多100位）       |
| cardBgc        | String  | 否       | 2 或 /oss/card/bg1.png                                       | 卡种背景图URL（数字或OSS路径，最多100位）      |
| cardTtype      | Integer | 是       | 1                                                            | 卡种类型：1-余额卡，2-次数卡，3-时效卡，4-积分卡 |
| presetRecharge | String  | 是       | [{"itemName":"充值100元","itemDesc":"首次充值赠送10元","amount":110.00}] | 预设充值项目（JSON字符串）                     |
| presetCost     | String  | 是       | [{"itemName":"洗车服务","itemDesc":"普通洗车一次","amount":30.00}] | 预设消费项目（JSON字符串）                     |
| autoNotify     | Integer | 是       | 1                                                            | 自动消息通知：0-关闭，1-短信，2-订阅，3-推送   |
| crossStore     | Integer | 是       | 0                                                            | 跨店通用：0-仅本店铺，1-同商家跨店通用         |

* **响应结果**：

| 字段名           | 类型    | 示例值                   | 描述                                     |
| ---------------- | ------- | ------------------------ | ---------------------------------------- |
| code             | Int     | 200                      | 状态码：200成功，40001无权限，40002店铺不存在 |
| data.cardTypeId  | Long    | 20001                    | 卡种 ID（自增主键）                      |
| data.storeId     | String  | uuid-90ab-cdef-1234-5678 | 所属店铺 ID                              |
| data.merchantId  | String  | uuid-5678-90ab-cdef-1234 | 所属商家 ID                              |
| data.cardTypeName| String  | 黄金余额卡               | 卡种名称                                 |
| data.cardTtype   | Integer | 1                        | 卡种类型                                 |
| data.createTime  | String  | 2025-10-24 10:00:00      | 创建时间（yyyy-MM-dd HH:mm:ss）          |
| message          | String  | 会员卡种创建成功         | 响应消息                                 |

* **响应示例**：

```json
{
  "code": 200,
  "data": {
    "cardTypeId": 20001,
    "storeId": "uuid-90ab-cdef-1234-5678",
    "merchantId": "uuid-5678-90ab-cdef-1234",
    "cardTypeName": "黄金余额卡",
    "cardTtype": 1,
    "createTime": "2025-10-24 10:00:00"
  },
  "message": "会员卡种创建成功"
}
```

### 4.6.2 接口 2：卡种列表查询

* **接口名**：卡种列表查询（查询指定店铺下所有会员卡种）

* **业务背景**：所有携带有效令牌的用户均可查询，用于展示店铺可办理的卡种；系统按创建时间倒序返回，支持分页。

* **认证**：任意有效令牌，仅校验令牌有效性

* **URL**：/api/v1/member-card-types/list-query

* **HTTP Method**：GET

* **请求头**：

| 请求头字段    | 是否必填 | 示例值            | 描述         |
| ------------- | -------- | ----------------- | ------------ |
| Authorization | 是       | Bearer {任意令牌} | 任意有效令牌 |

* **请求参数**（查询参数）：

| 参数名       | 类型    | 是否必填 | 示例值                   | 描述                                             |
| ------------ | ------- | -------- | ------------------------ | ------------------------------------------------ |
| storeId      | String  | 是       | uuid-90ab-cdef-1234-5678 | 目标店铺 ID                                      |
| cardTtype    | Integer | 否       | 1                        | 卡种类型筛选：1-余额卡，2-次数卡，3-时效卡，4-积分卡 |
| cardTypeName | String  | 否       | 余额卡                   | 卡种名称模糊筛选                                 |
| crossStore   | Integer | 否       | 0                        | 跨店通用筛选：0-仅本店铺，1-同商家跨店           |
| pageNum      | Integer | 是       | 1                        | 页码（从1开始）                                  |
| pageSize     | Integer | 是       | 10                       | 每页条数（1-50）                                 |

* **响应结果**：

| 字段名                        | 类型    | 示例值                   | 描述                               |
| ----------------------------- | ------- | ------------------------ | ---------------------------------- |
| code                          | Int     | 200                      | 状态码                             |
| data.total                    | Long    | 2                        | 卡种总数                           |
| data.pageNum                  | Int     | 1                        | 当前页码                           |
| data.pageSize                 | Int     | 10                       | 每页条数                           |
| data.list                     | Array   | -                        | 卡种列表                           |
| data.list[].cardTypeId        | Long    | 20001                    | 卡种 ID                            |
| data.list[].cardTypeName      | String  | 黄金余额卡               | 卡种名称                           |
| data.list[].cardTtype         | Integer | 1                        | 卡种类型                           |
| data.list[].cardTtypeName     | String  | 余额卡                   | 卡种类型名称                       |
| data.list[].description       | String  | 消费满100减10            | 卡种描述                           |
| data.list[].autoNotify        | Integer | 1                        | 自动消息通知类型                   |
| data.list[].crossStore        | Integer | 0                        | 跨店通用设置                       |
| data.list[].cardMask          | String  | 1                        | 卡面蒙版图片                       |
| data.list[].cardBgc           | String  | 2                        | 卡种背景图URL                      |
| data.list[].presetRechargeCount | Integer | 2                      | 预设充值项目数量                   |
| data.list[].presetCostCount   | Integer | 3                        | 预设消费项目数量                   |
| message                       | String  | 会员卡种列表查询成功     | 响应消息                           |

* **认证**：任意有效令牌，仅校验令牌有效性

* **URL**：/api/v1/member-card-types/detail-query

* **HTTP Method**：GET

* **请求头**：

| 请求头字段    | 是否必填 | 示例值            | 描述         |

### 4.6.4 接口 4：会员卡种修改设置

* **接口名**：会员卡种修改设置（修改卡种基础信息）

* **业务背景**：商家或店长修改卡种的基础信息（卡种名称、描述、预设项目、通知类型、跨店通用），卡种类型一旦创建不可修改。

* **认证**：
  - 商家：可使用普通令牌（tokenType=1，role=MERCHANT）
  - 店长：必须使用工作令牌（tokenType=3，role=manager）

* **URL**：/api/v1/member-card-types/set

* **HTTP Method**：POST

* **请求头**：

| 请求头字段    | 是否必填 | 示例值                      | 描述                       |
| ------------- | -------- | --------------------------- | -------------------------- |
| Content-Type  | 是       | application/json            | 请求体数据格式             |
| Authorization | 是       | Bearer {普通令牌/工作令牌}  | 商家普通令牌或店长工作令牌 |

* **请求参数**（请求体）：

| 参数名         | 类型    | 是否必填 | 示例值                                                       | 描述                                         |
| -------------- | ------- | -------- | ------------------------------------------------------------ | -------------------------------------------- |
| cardTypeId     | Long    | 是       | 20001                                                        | 卡种 ID                                      |
| storeId        | String  | 是       | uuid-90ab-cdef-1234-5678                                     | 所属店铺 ID                                  |
| cardTypeName   | String  | 否       | 黄金余额卡Pro                                                | 新卡种名称（2-50位，不填则不修改）           |
| description    | String  | 否       | 消费满200减20                                                | 新卡种描述（0-500位，不填则不修改）          |
| cardMask       | String  | 否       | 1 或 /oss/card/mask1.png                                     | 新卡面蒙版图片（不填则不修改）               |
| cardBgc        | String  | 否       | 2 或 /oss/card/bg1.png                                       | 新卡种背景图URL（不填则不修改）              |
| presetRecharge | String  | 否       | [{"itemName":"充值200元","itemDesc":"赠送30元","amount":230}] | 新预设充值项目（不填则不修改）               |
| presetCost     | String  | 否       | [{"itemName":"精洗服务","itemDesc":"精细洗车","amount":50}]   | 新预设消费项目（不填则不修改）               |
| autoNotify     | Integer | 否       | 2                                                            | 新自动消息通知类型（0-3，不填则不修改）      |
| crossStore     | Integer | 否       | 1                                                            | 新跨店通用设置（0-1，不填则不修改）          |

* **响应结果**：

| 字段名             | 类型   | 示例值                         | 描述                     |
| ------------------ | ------ | ------------------------------ | ------------------------ |
| code               | Int    | 200                            | 状态码                   |
| data.cardTypeId    | Long   | 20001                          | 卡种 ID                  |
| data.updateTime    | String | 2025-10-24 11:30:00            | 修改时间                 |
| data.updatedFields | Array  | ["cardTypeName", "autoNotify"] | 已更新的字段列表         |
| message            | String | 会员卡种修改成功               | 响应消息                 |

* **响应示例**：

```json
{
  "code": 200,
  "data": {
    "cardTypeId": 20001,
    "updateTime": "2025-10-24 11:30:00",
    "updatedFields": ["cardTypeName", "description", "autoNotify"]
  },
  "message": "会员卡种修改成功"
}
```
