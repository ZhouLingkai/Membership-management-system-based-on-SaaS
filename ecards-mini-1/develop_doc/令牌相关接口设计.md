## 令牌相关知识
| 令牌类型           | 说明                                                                   |
| -------------- | -------------------------------------------------------------------- |
| 用户普通令牌         | 上线后获取普通令牌，2 小时有效，可进行无风险的接口调用。如查询，个人信息修改。                             |
| 用户特权令牌         | 普通用户出示二维码时，二维码由特级令牌构成，5 分钟内有效，可进行办卡，充值，消费，绑定为店内店员的操作。                |
| 员工工作令牌         | 商家、店员、员工进入工作界面后（必须是指定店铺），有效时间为 1 小时                                  |
| 商家权限令牌         | 商家进行需要输入二级密码的操作时，会获取权限令牌，5 分钟内有效，可以进行绑定店内店员的操作，修改，删除的操作（5 分钟内最多 3 次） |
| 自动登录令牌 /长期身份凭证 | 仅用于触发 “自动登录”（换取普通令牌），不直接用于业务接口调用，有效期 7 天，支持动态续期，需纳入黑名单管理。            |
注意以下X-Device-ID实际是Device-ID，以Device-ID为准

### 2.1 接口1：获取普通令牌

#### 2.1.1 基础信息
- **请求地址**：`/api/v1/tokens/normal`
- **请求方法**：`POST`
- **权限说明**：无需认证（公开接口）
- **接口简介**：支持三种登录方式获取普通令牌：验证码登录（loginType=1）、密码登录（loginType=2）、自动登录（loginType=3）

#### 2.1.2 请求参数

**请求头（Header）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| X-Device-ID | String | 是 | 设备唯一标识 |

**请求体（Body）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| phone | String | 条件 | 手机号（AES-CBC加密后），loginType=1/2时必填 |
| verifyCode | String | 条件 | 短信验证码（6位数字），loginType=1时必填 |
| password | String | 条件 | 密码（明文），loginType=2时必填 |
| loginType | Integer | 是 | 登录方式：1-验证码登录，2-密码登录，3-自动登录 |
| autoLoginToken | String | 条件 | 自动登录凭证，loginType=3时必填 |
| platform | String | 是 | 平台类型：MINI_PROGRAM/WEB |

**返回参数**
| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | String | 普通令牌（JWT） |
| expireTime | String | 令牌过期时间（yyyy-MM-dd HH:mm:ss） |
| jti | String | 令牌唯一标识 |
| userRole | String | 用户角色 |
| autoExpireTime | String | 自动登录令牌过期时间（可选） |

### 2.2 接口2：获取特权令牌

#### 2.2.1 基础信息
- **请求地址**：`/api/v1/tokens/privilege`
- **请求方法**：`POST`
- **权限说明**：需要携带普通令牌（Authorization: Bearer xxx）
- **接口简介**：用于申请临时性高危操作权限令牌（单次有效）

#### 2.2.2 请求参数

**请求头（Header）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Authorization | String | 是 | Bearer {普通令牌} |
| X-Device-ID | String | 是 | 设备唯一标识 |

**请求体（Body）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| targetOperate | List\<String\> | 是 | 目标操作列表：CARD_CREATE/RECHARGE/CONSUME/STAFF_BIND/APPOINTMENT |

**返回参数**
| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | String | 特权令牌（JWT） |
| expireTime | String | 令牌过期时间（yyyy-MM-dd HH:mm:ss） |
| jti | String | 令牌唯一标识 |
| singleUse | Boolean | 是否单次有效（固定为true） |

---

### 2.3 接口3：获取工作令牌

#### 2.3.1 基础信息
- **请求地址**：`/api/v1/tokens/work`
- **请求方法**：`POST`
- **权限说明**：需要携带普通令牌（Authorization: Bearer xxx）
- **接口简介**：员工用户申请工作店铺的操作权限令牌

#### 2.3.2 请求参数

**请求头（Header）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Authorization | String | 是 | Bearer {普通令牌} |
| X-Device-ID | String | 是 | 设备唯一标识 |

**请求体（Body）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| storeId | String | 是 | 工作店铺ID（UUID字符串） |

**返回参数**
| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | String | 工作令牌（JWT） |
| expireTime | String | 令牌过期时间（yyyy-MM-dd HH:mm:ss） |
| jti | String | 令牌唯一标识 |
| storeId | String | 绑定的工作店铺ID |
| merchantId | String | 店铺所属商家ID |

### 2.4 接口4：获取管理令牌

#### 2.4.1 基础信息
- **请求地址**：`/api/v1/tokens/manager`
- **请求方法**：`POST`
- **权限说明**：需要携带普通令牌（Authorization: Bearer xxx）
- **接口简介**：商家用户通过二级密码申请管理令牌（5分钟内最多使用3次）

#### 2.4.2 请求参数

**请求头（Header）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Authorization | String | 是 | Bearer {普通令牌} |
| X-Device-ID | String | 是 | 设备唯一标识 |

**请求体（Body）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| merchantId | String | 是 | 商家ID（UUID字符串） |
| sndPswd | String | 是 | 二级密码（明文） |
| storeId | String | 否 | 目标操作店铺ID（操作店铺时必填） |

**返回参数**
| 参数名 | 类型 | 说明 |
|--------|------|------|
| token | String | 管理令牌（JWT） |
| expireTime | String | 令牌过期时间（yyyy-MM-dd HH:mm:ss） |
| jti | String | 令牌唯一标识 |
| maxUseCount | Integer | 5分钟内最大使用次数（固定为3） |
| usedCount | Integer | 已使用次数（初始为0） |

---

### 2.5 接口5：刷新令牌

#### 2.5.1 基础信息
- **请求地址**：`/api/v1/tokens/refresh`
- **请求方法**：`PUT`
- **权限说明**：需要携带旧令牌（Authorization: Bearer xxx）
- **接口简介**：刷新普通令牌（tokenType=1）或工作令牌（tokenType=2），旧令牌自动加入黑名单

#### 2.5.2 请求参数

**请求头（Header）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Authorization | String | 是 | Bearer {旧令牌} |
| X-Device-ID | String | 是 | 设备唯一标识 |

**请求体（Body）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| tokenType | Integer | 是 | 令牌类型：1-普通令牌，2-工作令牌 |

**返回参数**
| 参数名 | 类型 | 说明 |
|--------|------|------|
| newToken | String | 刷新后的新令牌（JWT） |
| newExpireTime | String | 新令牌过期时间（yyyy-MM-dd HH:mm:ss） |
| oldJti | String | 旧令牌JTI（已自动加入黑名单） |
| newJti | String | 新令牌JTI |

---

### 2.6 接口6：注销令牌

#### 2.6.1 基础信息

- **请求方法**：`DELETE`
- **权限说明**：需要携带令牌（Authorization: Bearer xxx）

#### 2.6.2 请求参数

**请求头（Header）**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Authorization | String | 是 | Bearer {令牌} |
| X-Device-ID | String | 是 | 设备唯一标识 |
