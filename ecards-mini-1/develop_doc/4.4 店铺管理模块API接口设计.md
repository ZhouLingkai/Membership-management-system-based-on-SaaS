# 会员管理系统 API 接口文档

## 1. 统一返回格式

所有接口均使用 "Result" 包装器返回数据，结构如下：

| 字段名  | 类型   | 描述                                        |
| ------- | ------ | ------------------------------------------- |
| code    | Int    | 状态码：200 成功，非 200 失败（含业务错误码） |
| data    | Object | 响应数据对象（成功时非空，失败时可空）       |
| message | String | 响应消息：成功时返回操作结果，失败时返回具体错误提示 |

## 4. 接口详情（续）

### 4.4 店铺管理模块

#### 4.4.1 接口列表

| 序号 | 接口名称           | URL                      | HTTP 方法 | 功能简述                                                                 |
| ---- | ------------------ | ------------------------ | --------- | ------------------------------------------------------------------------ |
| 1    | 店铺创建           | /api/v1/stores           | POST      | 商户创建关联店铺，生成店铺 ID                                            |
| 2    | 店铺信息查询（单个） | /api/v1/stores/{storeId} | GET       | 商户查询单个店铺的详细信息                                               |
| 3    | 店铺信息查询（列表） | /api/v1/stores           | GET       | 商户查询所有关联店铺信息                                                 |
| 4    | 店铺基础信息修改   | /api/v1/stores/{storeId} | PUT       | 商户修改店铺信息（名称、地址、营业时间、建店时间、门头店照、联系电话、联系微信号、营业执照、是否能预约等） |

#### 4.4.2 接口 1：店铺创建

* **业务背景**：已审核通过的商户创建关联店铺，需指定店铺基础信息，店铺创建后默认启用，可关联员工执行日常操作。

* **认证**：需携带商户管理令牌；校验令牌有效性、角色（仅 MERCHANT）；校验商户状态（审核通过）

* **URL**：/api/v1/stores

* **HTTP Method**：POST

* **请求头**：同 "商户资质上传 / 更新" 请求头，需携带 `X-Request-ID` 用于幂等性校验

* **请求参数**：

| 参数名          | 类型           | 是否必填 | 示例值                   | 描述                                   |
| --------------- | -------------- | -------- | ------------------------ | -------------------------------------- |
| merchantId      | String         | 是       | uuid-5678-90ab-cdef-1234 | 商户 ID（与令牌一致）                  |
| storeName       | String         | 是       | XX便利店（朝阳店）       | 店铺名称（长度 1-100 位）              |
| storeType       | String         | 否       | CONVENIENCE              | 店铺类型（长度不超过50字符）           |
| storeAddress    | String         | 否       | 北京市朝阳区XX路XX号     | 店铺地址（明文，长度不超过255字符）    |
| contactPhone    | String         | 是       | 13912345678              | 联系电话（明文，长度不超过20字符）     |
| contactWx       | String         | 是       | wxid_xxx                 | 联系微信号（明文，长度不超过20字符）   |
| businessHours   | String         | 否       | 08:00-22:00              | 营业时间（长度不超过100字符）          |
| storePhotos     | String         | 是       | stores/xxx/photo.jpg     | 门头店照（OSS对象路径，长度不超过255字符） |
| businessLicense | String         | 是       | stores/xxx/license.jpg   | 营业执照（OSS对象路径，长度不超过255字符） |
| appointment     | Integer        | 否       | 1                        | 是否支持预约：1-开启，0-关闭（默认0）  |
| openStoreTime   | LocalDateTime  | 否       | 2020-01-01T00:00:00      | 建店时间                               |

* **响应结果**：

| 字段名           | 类型   | 示例值                   | 描述                                     |
| ---------------- | ------ | ------------------------ | ---------------------------------------- |
| code             | Int    | 200                      | 状态码：200 成功，403 - 商户状态异常     |
| data.storeId     | String | uuid-cdef-1234-5678-90ab | 店铺 ID（UUID 字符串）                   |
| data.merchantId  | String | uuid-5678-90ab-cdef-1234 | 关联商户 ID                              |
| data.storeStatus | String | ENABLED                  | 店铺状态：ENABLED - 启用，DISABLED - 禁用 |
| data.createTime  | String | 2025-10-23 10:00:00      | 店铺创建时间                             |
| message          | String | 店铺创建成功             | 响应消息（失败时返回具体错误信息）       |

* **响应示例**：

```json
{
  "code": 200,
  "data": {
    "storeId": "uuid-cdef-1234-5678-90ab",
    "merchantId": "uuid-5678-90ab-cdef-1234",
    "storeStatus": "ENABLED",
    "createTime": "2025-10-23 10:00:00"
  },
  "message": "店铺创建成功"
}
```

#### 4.4.3 接口 2：店铺信息查询（单个）

* **业务背景**：商户查询单个店铺的详细信息，包括基础信息、状态、创建时间等，用于店铺管理操作。

* **认证**：需携带商户普通令牌；校验令牌有效性、角色（仅 MERCHANT）；校验店铺归属（商户 ID 与店铺关联商户 ID 一致）

* **URL**：/api/v1/stores/{storeId}（storeId 为路径参数）

* **HTTP Method**：GET

* **请求头**：同 "商户注册" 请求头

* **请求参数（Query 参数）**：

| 参数名     | 类型   | 是否必填 | 示例值                   | 描述                  |
| ---------- | ------ | -------- | ------------------------ | --------------------- |
| merchantId | String | 是       | uuid-5678-90ab-cdef-1234 | 商户 ID（用于权限校验） |

* **响应结果**：

| 字段名              | 类型    | 示例值                   | 描述                                 |
| ------------------- | ------- | ------------------------ | ------------------------------------ |
| code                | Int     | 200                      | 状态码：200 成功，404 - 店铺不存在/无权限 |
| data.storeId        | String  | uuid-cdef-1234-5678-90ab | 店铺 ID                              |
| data.merchantId     | String  | uuid-5678-90ab-cdef-1234 | 关联商户 ID                          |
| data.storeName      | String  | XX便利店（朝阳店）       | 店铺名称                             |
| data.storeType      | String  | CONVENIENCE              | 店铺类型                             |
| data.storeAddress   | String  | 北京市朝阳区XX路XX号     | 店铺地址                             |
| data.contactPhone   | String  | 13912345678              | 联系电话                             |
| data.contactWx      | String  | wxid_xxx                 | 联系微信号                           |
| data.businessHours  | String  | 08:00-22:00              | 营业时间                             |
| data.storePhotos    | String  | stores/xxx/photo.jpg     | 门头店照（OSS对象路径）              |
| data.businessLicense| String  | stores/xxx/license.jpg   | 营业执照（OSS对象路径）              |
| data.appointment    | Integer | 1                        | 是否支持预约：1-开启，0-关闭         |
| data.storeStatus    | Integer | 1                        | 店铺状态（1-正常营业等）             |
| data.openStoreTime  | String  | 2020-01-01 00:00:00      | 建店时间                             |
| data.createTime     | String  | 2025-10-23 10:00:00      | 创建时间                             |
| data.lastUpdateTime | String  | 2025-10-23 10:00:00      | 最后更新时间                         |
| message             | String  | 店铺信息查询成功         | 响应消息（失败时返回具体错误信息）   |

* **响应示例**：

```json
{
  "code": 200,
  "data": {
    "storeId": "uuid-cdef-1234-5678-90ab",
    "merchantId": "uuid-5678-90ab-cdef-1234",
    "storeName": "XX便利店（朝阳店）",
    "storeType": "CONVENIENCE",
    "storeAddress": "北京市朝阳区XX路XX号",
    "contactPhone": "13912345678",
    "contactWx": "wxid_xxx",
    "businessHours": "08:00-22:00",
    "storePhotos": "stores/xxx/photo.jpg",
    "businessLicense": "stores/xxx/license.jpg",
    "appointment": 1,
    "storeStatus": 1,
    "openStoreTime": "2020-01-01 00:00:00",
    "createTime": "2025-10-23 10:00:00",
    "lastUpdateTime": "2025-10-23 10:00:00"
  },
  "message": "店铺信息查询成功"
}
```

#### 4.4.4 接口 3：店铺信息查询（列表）

* **业务背景**：商户查询所有关联店铺的基础信息（简化版），用于批量管理。

* **认证**：需携带商户普通令牌；校验令牌有效性、角色（仅 MERCHANT）

* **URL**：/api/v1/stores

* **HTTP Method**：GET

* **请求头**：同 "商户注册" 请求头

* **请求参数（Query 参数）**：

| 参数名     | 类型   | 是否必填 | 示例值                   | 描述                  |
| ---------- | ------ | -------- | ------------------------ | --------------------- |
| merchantId | String | 是       | uuid-5678-90ab-cdef-1234 | 商户 ID（与令牌一致） |

* **响应结果**：

| 字段名                | 类型    | 示例值                   | 描述                               |
| --------------------- | ------- | ------------------------ | ---------------------------------- |
| code                  | Int     | 200                      | 状态码：200 成功                   |
| data.total            | Int     | 2                        | 总店铺数量                         |
| data.list[].storeId   | String  | uuid-cdef-1234-5678-90ab | 店铺 ID（列表项）                  |
| data.list[].storeName | String  | XX便利店（朝阳店）       | 店铺名称（列表项）                 |
| data.list[].storeType | String  | CONVENIENCE              | 店铺类型（列表项）                 |
| data.list[].storeStatus | Integer | 1                      | 店铺状态（列表项）                 |
| data.list[].createTime | String | 2025-10-23 10:00:00      | 创建时间（列表项）                 |
| message               | String  | 店铺列表查询成功         | 响应消息（失败时返回具体错误信息） |

* **响应示例**：

```json
{
  "code": 200,

        "storeName": "XX便利店（朝阳店）",
        "storeType": "CONVENIENCE",
        "storeStatus": 1,
        "createTime": "2025-10-23 10:00:00"
      },
      {
        "storeId": "uuid-1234-5678-90ab-cdef",
        "storeName": "XX便利店（海淀店）",
        "storeType": "CONVENIENCE",
        "storeStatus": 1,
        "createTime": "2025-10-24 09:00:00"
      }
    ]
  },
  "message": "店铺列表查询成功"

```

#### 4.4.5 接口 4：店铺信息修改

* **业务背景**：商户修改店铺信息（名称、地址、联系电话、联系微信、营业时间、门头店照、营业执照、是否支持预约、建店时间）。

* **认证**：需携带商户普通令牌；校验令牌有效性、角色（仅 MERCHANT）；校验店铺归属

* **URL**：/api/v1/stores/{storeId}（storeId 为路径参数）

* **HTTP Method**：PUT

* **请求头**：同 "商户注册" 请求头

* **请求参数**：

| 参数名          | 类型          | 是否必填 | 示例值                   | 描述                                     |
| --------------- | ------------- | -------- | ------------------------ | ---------------------------------------- |
| merchantId      | String        | 是       | uuid-5678-90ab-cdef-1234 | 商户 ID（与令牌一致）                    |
| storeName       | String        | 否       | XX便利店（朝阳新馆）     | 新店铺名称（长度1-100字符，不填则不修改） |
| storeAddress    | String        | 否       | 北京市朝阳区XX路XX号     | 新店铺地址（长度不超过255字符，不填则不修改） |
| contactPhone    | String        | 否       | 13987654321              | 新联系电话（明文，长度不超过20字符，不填则不修改） |
| contactWx       | String        | 否       | wxid_new                 | 新联系微信号（长度不超过20字符，不填则不修改） |
| businessHours   | String        | 否       | 07:30-23:00              | 新营业时间（长度不超过100字符，不填则不修改） |
| storePhotos     | String        | 否       | stores/xxx/new_photo.jpg | 新门头店照（OSS对象路径，不填则不修改）  |
| businessLicense | String        | 否       | stores/xxx/new_license.jpg | 新营业执照（OSS对象路径，不填则不修改） |
| appointment     | Integer       | 否       | 1                        | 是否支持预约：1-开启，0-关闭（不填则不修改） |
| openStoreTime   | LocalDateTime | 否       | 2020-01-01T00:00:00      | 新建店时间（不填则不修改）               |

* **响应结果**：

| 字段名          | 类型   | 示例值              | 描述                               |
| --------------- | ------ | ------------------- | ---------------------------------- |
| code            | Int    | 200                 | 状态码：200 成功，403 - 无店铺权限 |
| data.updateTime | String | 2025-10-23 14:30:00 | 信息更新时间                       |
| message         | String | 店铺信息修改成功    | 响应消息（失败时返回具体错误信息） |

* **响应示例**：

```json
{
  "code": 200,
  "data": {
    "updateTime": "2025-10-23 14:30:00"
  },
  "message": "店铺信息修改成功"
}
```
