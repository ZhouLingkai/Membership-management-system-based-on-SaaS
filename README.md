# EcardsV1 · 基于 SaaS 的会员管理系统

> 设计并实现了一套面向中小商家的多租户 SaaS 会员管理平台，涵盖 Spring Boot 后端、微信小程序商家端与会员端、腾讯云开发后端，支持会员卡管理、积分消费、预约服务、多角色权限等核心业务。
> 独立全栈开发者，具备SpringBoot开发经验与前端开发经验，配合AI Coding + 人工审查能力快速开发，开发时间2025年11月~2026年1月。
---

## 项目概览

本项目采用**三层解耦**架构：

```
腾讯云开发后端（鉴权 / NoSQL）
        ↕ HTTP
微信小程序前端（商家端 + 会员端）
        ↕ HTTP
Spring Boot 后端（核心业务 / MySQL / Redis / 阿里云 OSS）
```

| 层级 | 技术栈 |
|------|--------|
| 核心后端 | Spring Boot 3.4 · Java 17 · Spring Security · JPA · MySQL · Redis · Redisson |
| 无服务器后端 | 腾讯云开发（CloudBase）· Node.js 云函数 · 云数据库 |
| 前端 | 微信小程序（原生）· 腾讯云 SDK |
| 存储 | 阿里云 OSS + STS 临时密钥 |
| 安全 | JWT（jjwt 0.12）· Argon2 密码哈希 · AES-256-GCM（BouncyCastle）|

## 功能清单

### Spring Boot 后端

- **用户体系**：平台管理员、商家、员工、会员四角色，基于 Spring Security 的接口权限控制
- **认证与令牌**：短信验证码注册/登录、账密登录、JWT 双 Token（登录令牌 + 工作令牌）、Redis Token 黑名单、设备绑定防跨设备调用
- **商家管理**：商家注册审核、商家信息维护、升级/降级管理
- **店铺管理**：多店铺创建与管理
- **员工管理**：员工账号创建、绑定店铺、权限管控（禁止删除操作）
- **会员卡管理**：余额会员卡、积分会员卡、时长会员卡，开卡/销卡/查询
- **会员卡类型管理**：自定义卡种配置
- **消费与积分**：消费记录、积分增减、交易流水统计
- **预约系统**：预约创建、预约资源管理、高级预约
- **图片存储**：阿里云 OSS + STS 临时密钥上传
- **操作日志**：管理员操作行为记录与查询
- **定时任务**：过期会员卡自动处理等

### 微信小程序前端（商家端）

- 商家登录 / 注册 / 个人信息管理
- 会员搜索与会员卡办理、充值、消费核销
- 店铺创建与管理
- 员工绑定与管理
- 会员卡类型配置
- 交易记录与统计报表
- 预约管理（资源配置、预约列表）

### 微信小程序前端（会员端）

- 微信手机号一键注册登录（商家 ID + 微信手机号）
- 我的会员卡列表与卡详情
- 积分 / 余额 / 消费记录查询
- 资源级预约服务

### 腾讯云开发后端（CloudBase 云函数）

- **令牌管理**：工作令牌兑换、令牌刷新、退出登录、特权令牌、会员二维码令牌
- **商家管理**：短信验证码、注册/登录、商家信息维护、密码修改与重置、二级密码管理
- **店铺管理**：店铺创建、信息查询与修改、预设配置、跨店组管理
- **员工管理**（待开发）
- **交易管理**（待开发）
- **预约管理**（待开发）

## 快速开始（仅供参考）

### 后端

```bash
cd member-management
# 配置 application.yml（见下方「环境变量与密钥」）
mvn spring-boot:run
```

**依赖服务**：MySQL 8+、Redis 6+

### 小程序

1. 使用微信开发者工具打开 `ecards-mini-1/` 目录
2. 在 `app.js` 中配置云环境 ID 及后端地址
3. 编译运行

### 云函数

在腾讯云控制台（CloudBase）中，逐个上传 `TCloudBase/cloudfunctions/` 下的各云函数目录。

## 环境变量与密钥配置

> ⚠️ **所有密钥、Token、连接串均不得硬编码，必须通过环境变量或配置中心注入。**

Spring Boot 后端需在 `application.yml`（生产环境请使用环境变量覆盖）中配置：

| 配置项 | 说明 |
|--------|------|
| `spring.datasource.url/username/password` | MySQL 连接信息 |
| `spring.data.redis.*` | Redis 连接信息 |
| `jwt.secret` | JWT 签名密钥（至少 256 bit） |
| `aes.key` | AES-256-GCM 加密密钥 |
| `aliyun.oss.accessKeyId/accessKeySecret` | 阿里云 OSS 密钥 |
| `aliyun.sts.roleArn` | 阿里云 STS 角色 ARN |
| `sms.secretId/secretKey` | 短信服务密钥 |

云函数需在腾讯云 CloudBase 环境变量中配置：

| 变量名 | 说明 |
|--------|------|
| `JWT_SECRET` | JWT 签名密钥 |
| `SMS_SECRET_ID / SMS_SECRET_KEY` | 短信服务密钥 |
| `ENV_ID` | 云开发环境 ID |
